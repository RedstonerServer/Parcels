# Parcels2

Plot management and world generator plugin inspired by [PlotMe](https://github.com/WorldCretornica/PlotMe-Core).

Newer version of discontinued [Parcels](https://github.com/RedstonerServer/Parcels).

Written in Kotlin.
This project is WIP.
